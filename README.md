# 🎸 BandGen

A full-stack band name generator built with Node.js, Express, and EJS. Users can generate random band names based on genre selection.

---

## 🚀 Live Demo
https://band-gen.onrender.com

---

## ✨ Features
- 🎵 Genre-based band name generation
- 🎲 Random adjective + noun combinations
- 📋 Copy generated name to clipboard
- 🔁 Generate again functionality
- 🎨 Styled responsive UI

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript

---

## 🎮 How It Works
1. Select a music genre
2. Click **Generate Name**
3. Receive a randomly generated band name
4. Copy it or generate another
